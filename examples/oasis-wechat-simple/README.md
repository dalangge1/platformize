# platformize-oasis-demo-wechat-simple

最基本的 oasis demo, 显示一个 gltf 模型，小程序包大小`552kb`

[platformize](https://github.com/deepkolos/platformize)

# 运行

请使用**最新版本**的微信开发工具打开，切换到自己的测试 appId 即可运行，**打开调试模式**

如果需要修改代码，可按照已下步骤构建

```sh
# 安装依赖
> rush install

# dev
> rush build:watch

# 小程序开发工具打开本目录即可

# prod
> rush build
```
